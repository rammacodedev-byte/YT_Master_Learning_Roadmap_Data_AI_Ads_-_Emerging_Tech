‚≠ê Types of Data Analysis - Data Prospect
- Qualitative (cannot be measured data) 
- Quantitative (can be measured data, text-based) 

‚≠ê Types Of Analysis Techniques
- Statistical
  - Descriptive (already happened)
  - Inferential (different sample collection for prediction)
  - Exploratory (find connections + hypotheses)
  - Diagnostic (focus on "why", prediction analysis)
  - Perspective Analysis
  - Recommender Analysis (suggestions for future problems)
  - Text Analysis 

‚≠ê Statistical Analysis
- Dashboard creation, graphs, models of data, customer data analysis
- Descriptive: summarize all data (numerical & categorical, full population)
- Inferential: sample data (e.g., 30% of population)
- Exploratory: store relationships/connections 

‚≠ê Diagnostic Analysis
- Graphs check reason for growth/loss
- Why did it happen? Identify issues
- Solve problems using past data 

‚≠ê Predictive Analysis
- Predict what is likely to happen
- Check budgets and reasons for buying (season, salary, etc.)
- Use past data, diagnostic and descriptive results
- ML/AI for future trends & problem prediction
- Forecasting examples: temperature, rainfall, population
- Accuracy depends on data 

‚≠ê Prescriptive Analysis
- Combine prediction data & decide actions
- Key areas: marketing, sales, customer experience, HR, finance, logistics 

‚≠ê Text Analysis
- Document comparison, check duplications, feedbacks
- Text mining for large data
- Sentiment analysis, topic identification, mood/tree analysis, toxic words
- Word-based analysis (Twitter, Facebook, Instagram)
- Survey responses, pain points, communication setup 

‚≠ê Techniques & Methods

‚≠ê Cluster Analysis
- Explore data patterns
- Homogeneous (similar) vs Heterogeneous (different) clusters
- Business perspective: sales, marketing, insurance, geology 

‚≠ê Cohort Analysis
- Subset behavior analysis
- Divide groups (example Gmail: spam vs important)
- Market analysis: online/offline purchasing trends 

‚≠ê Regression Analysis
- Dependent/independent variable analysis
- Linear & non-linear regression
- Identify patterns for past & future predictions 

‚≠ê Factor Analysis
- Dimension reduction
- Identify hidden patterns
- Summarize variables based on correlation & covariance 

‚≠ê Neural Networks

- Multi-layer networks
- Applications: image recognition, handwriting, speech, signals, language, object recognition, AI generation, stock prediction, text classification 

‚≠ê Data Mining
- Explore relations, patterns, trends
- Example: KPIs, intelligent alerts 

‚≠ê Text Analytics / Text Mining

‚≠ê Time Series Analytics

‚≠ê Decision Trees
